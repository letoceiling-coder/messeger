Здравствуйте!

Нужна помощь с запуском Node.js приложения и настройкой Apache.

Сайт: parser-auto.siteaccess.ru
Приложение: NestJS Backend

ПРОБЛЕМЫ:

1. Нет прав для использования портов (3000, 30000)
   Ошибка: Error: listen EPERM: operation not permitted

2. Apache не проксирует запросы к Backend
   API не доступен через https://parser-auto.siteaccess.ru/api/

ЧТО НУЖНО:

1. Разрешить использование порта для Node.js приложения
   (например, порт 30000 или другой доступный порт)

2. Включить модули Apache:
   - mod_proxy
   - mod_proxy_http
   - mod_rewrite

3. Настроить проксирование:
   - /api → http://localhost:PORT/api
   - /socket.io → http://localhost:PORT/socket.io

ЧТО УЖЕ СДЕЛАНО:

- Файл .htaccess создан в public_html/
- Файл .nginx.conf создан в корне сайта
- Backend готов к запуску

Помогите, пожалуйста, настроить.
