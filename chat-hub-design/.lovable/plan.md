

# План визуальных улучшений мессенджера

## Обзор

Проект уже имеет хорошую базовую структуру мессенджера в стиле Telegram. Предлагаю комплексные визуальные улучшения для всех основных экранов: список чатов, чаты, настройки, боты, каналы и лента.

---

## 1. Исправление ошибки сборки

Перед улучшениями нужно исправить отсутствующий импорт на странице контактов.

**Файл:** `src/pages/ContactsPage.tsx`

Добавить импорт:
```typescript
import EmptyState from '@/components/common/EmptyState';
```

### 2.1 Визуальные карточки чатов
- Добавить тонкие тени для разделения чатов
- Улучшить выделение закреплённых чатов (градиентный фон)
- Добавить иконки для ботов и каналов в списке
- Визуальный индикатор "печатает..." с анимированными точками

### 2.2 Улучшенный header

- Красивая иллюстрация когда нет чатов
