# РЕШЕНИЕ ПРОБЛЕМЫ ПОДКЛЮЧЕНИЯ

## ПРОБЛЕМА

```
ssh: Could not resolve hostname parser-auto.site-access.ru
```

Домен не резолвится для SSH. Это нормально - домен может быть настроен только для веб-доступа.

---

## РЕШЕНИЕ: НАЙТИ IP АДРЕС СЕРВЕРА

### Способ 1: В панели Beget (самый простой)

1. Войдите в панель управления Beget
2. Найдите раздел:
   - **"Информация о сервере"**
   - **"SSH доступ"**
   - **"Сервер"**
   - **"Доступ"**
3. Найдите **IP адрес сервера** (например: `123.45.67.89`)
4. Используйте IP для подключения:

```bash
ssh dsc23ytp@123.45.67.89
```

---

### Способ 2: На самом сервере

Если вы уже подключены к серверу (через панель Beget), выполните:

```bash
hostname -I
```

Или:

```bash
ip addr show | grep "inet " | grep -v 127.0.0.1
```

Запишите IP адрес и используйте его.

---

### Способ 3: Автоматическая настройка

После того как узнаете IP адрес, запустите:

```powershell
.\setup-ssh-with-ip.ps1
```

Введите IP адрес, когда спросит.

---

## НАСТРОЙКА SSH CONFIG

После того как узнаете IP адрес:

1. Откройте файл: `C:\Users\ВашеИмя\.ssh\config`
2. Добавьте (замените IP_АДРЕС на реальный):

```
Host dragon
    HostName IP_АДРЕС_СЕРВЕРА
    User dsc23ytp
    IdentityFile C:\Users\ВашеИмя\.ssh\id_rsa
```

3. Сохраните файл

После этого можно подключаться: `ssh dragon`

---

## ОБНОВИТЬ СКРИПТЫ

После того как узнаете IP адрес, обновите:

**`scripts/sync-to-server.ps1`** - замените:
```powershell
$SERVER_ADDRESS = "parser-auto.site-access.ru"
```
На:
```powershell
$SERVER_ADDRESS = "IP_АДРЕС_СЕРВЕРА"
```

---

## ПРОВЕРКА

После настройки:

```bash
ssh dsc23ytp@IP_АДРЕС
```

Или если настроен config:

```bash
ssh dragon
```

Если подключилось без пароля - отлично! ✅

## АЛЬТЕРНАТИВНЫЕ АДРЕСА BEGET

Beget может предоставлять специальные SSH адреса:

- `ssh.beget.com`
- `dragon.beget.com`
- `dsc23ytp@ssh.beget.com`

Проверьте в панели Beget, может быть указан специальный SSH адрес.
