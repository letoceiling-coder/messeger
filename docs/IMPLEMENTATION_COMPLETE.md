# Реализация завершена — что протестировать

## Реализовано

### 1. Отправка сообщений
- **POST /messages** — текстовые сообщения с ответом (replyToId)
- ChatPage: отправка, редактирование, удаление через API

### 2. Контакты
- **GET /users** — список пользователей (контакты)
- **GET /users/search?q=** — поиск по username/email
- **POST /chats/direct** — создать личный чат с пользователем
- ContactsPage: клик по контакту → создание чата и переход
- AddContactSheet: поиск пользователя → выбор → создание чата

### 3. WebSocket
- Подключение с JWT при авторизации
- **message:received** — новые сообщения в реальном времени
- **message:deleted** — удаление сообщений
- **message:edited** — редактирование
- **typing:start / typing:stop** — индикатор «печатает»

### 4. Удаление сообщений
- **DELETE /messages/:id** — удалить у себя
- **POST /messages/delete-for-everyone** — удалить у всех

### 5. Групповые чаты
- **POST /chats/group** — создание группы (name, memberIds, description)
- CreateChannelPage: создание канала/группы через API

## Как тестировать

### Авторизация
1. Открыть https://neekloai.ru (или http://89.169.39.244)
2. Вкладка «Телефон» или «Email» (зависит от AUTH_MODE)
3. Ввести номер/email → получить код → ввести код → вход

### Чаты
1. После входа — список чатов из API (если есть)
2. Если пусто — перейти в «Контакты»

### Контакты
1. Список — все пользователи (кроме текущего)
2. Кнопка «+» — поиск по username/email
3. Клик по контакту — создание чата и переход в него

### Сообщения
1. Ввод текста → Enter или кнопка отправки
2. Ответ на сообщение (long press → Ответить)
3. Редактирование (long press → Редактировать)
4. Удаление (long press → Удалить → у себя / у всех)

### Канал/Группа
1. Профиль → Создать канал
2. Название, описание → Создать
3. Переход в новый чат

### WebSocket (два окна)
1. Войти в двух вкладках/устройствах под разными пользователями
2. Создать личный чат между ними
3. Отправить сообщение — оно должно появиться у второго пользователя без обновления страницы
